CV Camera Utility

    - This is a basic cv recognition camera program written in Python. 

    - There are two different detector variations, a template and an orb detector. 
        - you can switch between the two by changing the 'detector' variable in main.py. (Make sure to change the text when calling show_frame())
    
    - I'm working on training it for simple objects now, like a soda can for the first iteration. Will work on human faces later. 
    
    - I'm planning to evolve this into a more specialized tool to configure DIY security cameras. 



    Not Implemented:

        - Footage and GUI casting to secondary device, such as tablets or standalone monitors. 

        - YOLO detector/ homgenous detector


    Dependencies: 
    
        - Python
        - OpenCV
        - numpy
        - matplotlib
